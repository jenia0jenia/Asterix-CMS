Вывод последних записей

В {preload} можно передавать любые условия, существующие в структуре записи. В том числе, можно передавать выборку по родительскому разделу.

Так как в структурах привязка к родителю может осуществляться по двум разным полям dep_path_dir (для списков) и dep_path_parent (для деревьев), для удобства вы можете просто передать параметр dir в качестве указателя на родителя.

Обратите внимание, связки деревьев делаются по полю SID, в то время как линейные структуры связываются по полю ID. В параметр DIR нужно передавать параметр с учётом этих знаний.
